RTS Camera

This is a small project I started in order to learn how to use UE5. The idea was that I wanted to recreate the camera system from the first Halo Wars game. I am continuing work on this whenever I get the time and it may develop into a game but for now I'm happy to use it as a learning tool for blueprints and C++.
I'm using the mapping contexts with Enhanced Input input actions to set up inputs for both controller and keyboard/mouse. Currently the camera is on a spring arm parented to an empty actor and the actor is the thing that's being moved around. The actor translates/rotates which brings the camera with it and the zooming uses a timeline to output the rotation and position the camera should have at each zoom level.
