DIY Saga

DIY Saga is a demo of a mobile game built for Android where the player completes tasks within a time limit to advance to the next level. Developed for a university assignment and built using Unity. The purpose of this project was to investigate the current popular games on the market and create a demo game demonstrating understanding of popular game styles, mechanics, and developer insights.

Recycle Michael

Recycle Michael was a Unity project for a second year university module where the aim was to design and make a mobile game focusing on monetization strategy and creating a good technical specification. The game I came up with was "Recycle Michael" in which you manage a recycling facility and clean up the city which is undergoing a trash crisis.
